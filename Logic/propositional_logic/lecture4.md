###symbols for logic
- Propositional Logic
    - $\neg$
    - $\rightarrow$
    - $\leftrightarrow$
    - $\vee$
    - $\land$
    - $\lor$
    - $\wedge$
    - $\vdash$
    - $\dashv$
- Predicate Logic
    - $\forall$
    - $\exists$
    - $\in$
    - $\models$
- Modal Logic
    - $\Box$
    - $\Diamond$


($\neg$ P $\wedge$ Q) $\rightarrow$ (P $\rightarrow$ R) is valid, hence
($\neg$(A $\vee$ B) $\wedge$ (B $\leftrightarrow$ C)) $\rightarrow$ ((A $\vee$ B) $\rightarrow$ (B $\leftrightarrow$ C))
