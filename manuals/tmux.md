- pane
    - **^a |**: new vertical
    - **^a -**: new horizontal
    - **^a x**: kill current

- scroll in buffers
    - ```^a [ ```: enter scroll mode
    - ```Fn + Pg Up or Pg Dn```: srcoll
    - ```q```: exit scroll mode

- seesions
    - ```^a c```: new session
    - ```^a l```: previous session
    - ```^a (```: pre
